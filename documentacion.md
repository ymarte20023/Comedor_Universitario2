/**
 * Proyecto: Sistema de control del inventario del Comedor universitario.
 
 * Descripción: Control de Inventario de Comedor Universitario: Gestión de proveedores, control de insumos perecederos, planificación de menús y alertas de stock crítico.
 * Fecha: 2026-02-09
 */

-----------------------------------------

 /**
 * Tecnologías:
 * - Backend: PHP.
 * - Base de Datos: MySQL o MariaDB.
 * - Control de Versiones : Git
 */
-----------------------------------------
/*
 * Estructura del Proyecto:
 * - app/
 *   - controllers/  (Controladores para manejar la lógica de negocio)
 *   - models/       (Modelos para interactuar con la base de datos)
 *   - views/        (Vistas para mostrar la interfaz de usuario)
 *   - core/         (Clases centrales como Router, Database, etc.)
 * - public/         (Archivo index.php y recursos públicos)
 * - config/         (Archivo de configuración)
 */

-----------------------------------------
/**
 * Funcionalidades:
 * - Gestión de Proveedores: CRUD para proveedores, historial de compras.
 
 * - Control de Insumos Perecederos: Registro de entradas/salidas, alertas de stock crítico.
 
 * - Planificación de Menús: Creación de menús semanales, asignación de insumos.

 * - Reportes: Inventario actual, consumo histórico, alertas de vencimiento.
 
 * - Autenticación: Sistema de login para usuarios autorizados.

-----------------------------------------
